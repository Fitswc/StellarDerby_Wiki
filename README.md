## 概述

这里是 基于游戏《Stellais》的《赛马娘 Pretty Derby》（**ウマ娘 プリティーダービー**）的同人Mod：《赛马娘：星海德比》（**Umamusume:StellarDerby**）的Wiki！本Wiki的建立初衷就是给游玩这款Mod的玩家一个官方攻略，以后就不会不知道怎么选啦！（虽然剧情也很线性）(虽然，还在施工)
