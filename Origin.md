# 起源

本Mod包含多个起源，每个起源代表着不同的背景以及玩法。

> 每个起源都允许在其他的星球开设分部

可选的起源如下

| 起源                                        | 加成                                                         | 玩法                                       |
| ------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------ |
| 三女神的光辉下                              | 工业区划+1；<br>添加地貌：三女神遗址<br>添加地貌：水晶胡萝卜矿脉；<br>添加建筑：基础赛场<br> | 探索三女神的真相，最基础的玩法             |
| ![CAF](origins_candleoff.png)烛尽光殇       | 其他国家对我方的评价:-1000<br>与其他文明建立联系时将会立刻宣战<br>对其他国家可使用全面战争理由"灭烛"<br>允许使用"光殇"轨道轰炸姿态<br>允许使用净化方式：残躯化火<br>获得国家修正：战争经济<br> | 强大的早战流派玩法，专注于战斗             |
| ![BUM](origin_uma_business_MAN.png)商业大师 | 商业协议效率：+40%<br>殖民地产生的帝国规模：+25%<br>渗透速度：+30%<br>破译：+2<br>允许在其他国家建立分部<br>允许对其他巨型企业政体国家使用：夺取商业主动权 战争借口<br> | 罪企玩法，将专注于以商业手段从内部瓦解对方 |

## 起源影响的剧情走向

烛尽光殇起源可能会导致部分剧情走向发生变化。

具体体现于：

1. 马娘堕落帝国对我们的态度

2.  马娘堕落帝国将不会与我们建立交流办公室

## 起源影响的建筑

商业大师起源带有的独特建筑：

1. 马娘涩图商城
2. 走私港
3. 马娘地下俱乐部
4. 地下武器制造厂
